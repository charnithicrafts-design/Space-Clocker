# Technology Stack: Space-Clocker

## Frontend Framework
- **React 19 (TypeScript):** Primary UI framework for building the reactive "Dopamine Fountain" interface.
- **Vite 6:** Lightning-fast build tool and development server, configured for port 3000.

## Styling & Animation
- **Tailwind CSS v4 (@tailwindcss/vite):** Utility-first CSS for "Cosmic Dark Mode" and "Ethereal & Translucent" components.
- **Framer Motion:** Powering high-fidelity micro-animations and physics-based interactions.

## Audio Engine
- **Web Audio API:** Custom SoundManager.ts utility for low-latency, organic audio feedback (pops, chimes, thuds).

## AI & Intelligence
- **Google Generative AI (@google/genai):** Integration with the Gemini API for smart features and reflection loops.

## Icons & Utilities
- **Lucide React:** Minimalist, consistent iconography.
- **clsx / tailwind-merge:** Utilities for dynamic class management.

## Environment & Tooling
- **TypeScript:** Ensuring type safety and surgical precision across the codebase.
- **Dotenv:** Managing sensitive configuration and API keys securely.

## Scripts & Automation
- 
pm run dev: Starts the development server on port 3000.
- 
pm run build: Generates the production build.
- 
pm run lint: Performs type-checking using 	sc.
