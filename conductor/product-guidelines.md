# Product Guidelines: Space-Clocker

## Prose Style: Trajectory-Professional
- **Tone:** Futuristic, punchy, and highly professional.
- **Voice:** Authoritative yet inspiring. Use terminology that resonates with advanced cloud architecture and career trajectories (e.g., "Trajectory", "Resonance", "Optimal Sync").
- **Clarity:** Keep instructions concise. Every word should serve a purpose, mimicking mission-critical communication.

## Branding & Visual Identity
### Color Palette
- **Optimal Dark Mode:** Primary background uses deep professional blues and purples to create depth and focus.
- **Neon Accents:** High-contrast cyan and magenta for interactive elements, highlights, and critical feedback.
- **Lab White/Silver:** Used for text and technical data overlays to ensure surgical precision and readability.

### Visual Tone: Ethereal & Translucent
- **Glassmorphism:** Extensive use of soft blurs and semi-transparent panels to create a layered, "heads-up display" (HUD) feel.
- **Glow & Depth:** Subtle outer glows for active components and deep shadows to establish hierarchy in a dark environment.

## UX Principles
### Total Feedback Loop
- Every user action (click, hover, scroll) must be met with a corresponding micro-animation (Framer Motion) and/or a subtle audio cue (Web Audio API).
- Feedback should be immediate and organic, reinforcing the "Dopamine Fountain" philosophy.

### High-Efficiency Navigation
- Minimize the number of steps to reach core functions (Macro Engine, Micro Engine, Reflection Loop).
- Use intuitive gestures and keyboard shortcuts where appropriate to streamline the user's workflow.

### Progressive Disclosure
- Avoid overwhelming the user with data. Reveal complexity only on demand (e.g., expanding goal details, viewing advanced metrics).
- Use tooltips and contextual overlays to provide information without cluttering the primary interface.

## Accessibility & Performance
- **Contrast:** Maintain high contrast ratios for text against dark backgrounds using the "Lab White" palette.
- **Latency:** Ensure all animations and audio triggers are low-latency to maintain the high-performance feel.
- **Responsive Design:** The interface must remain functional and visually striking across all screen sizes, prioritizing the "mobile HUD" experience.
