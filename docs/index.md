## Project Overview

Space-Clocker is a high-fidelity, space-themed productivity and skill-tracking application. It features a modular interface with Dashboard, Nebula, Orbit, and Stellar Matrix modes, utilizing glassmorphism aesthetics and rich animations.

## Quick Reference

- **Tech Stack:** React 19, Vite 6, Tailwind CSS 4, Framer Motion, Zustand
- **Entry Point:** `src/main.tsx` (App) | `src/db/db.worker.ts` (Database Worker)
- **Architecture Pattern:** Component-based Frontend with Local Browser Database
- **Database:** PGlite (Postgres-in-the-browser)
- **Deployment:** Vercel (Production) / PWA (Offline Support)

## Getting Started

### Prerequisites

- Node.js (Latest LTS)
- Google Gemini API Key (stored in `.env.local`)

### Setup

```bash
npm install
```

### Run Locally

```bash
npm run dev
```

### Run Tests

```bash
npm run test
```
